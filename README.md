# SMM Platform

**Единая платформа продвижения турагентства в Instagram и на сайте — с AI-агентом, аналитикой и приёмом заявок.**

[English version](README.en.md)

## Зачем

У турагентства каналы разрознены: Instagram, директ, комментарии, сайт, мессенджеры — всё вручную и в разных местах. SMM Platform сводит это в одну панель: подключаешь Instagram официальным входом Meta, платформа сама следит за доступом, отвечает клиентам по сценариям и подсказывает, что делать дальше — на основе реальных данных, а не догадок.

Два ключевых поинта: **(1)** консультация и сбор заявок, **(2)** продвижение и рост аккаунта. Всё остальное работает на эти две задачи.

## Возможности

- **Панель управления** — вход по логину и паролю, роли (администратор/менеджер), создание и деактивация пользователей, смена собственного пароля.
- **Подключение Instagram** — официальный OAuth-вход (Instagram API with Instagram Login, без пароля от аккаунта в открытом виде); токен доступа хранится зашифрованным (AES-256-GCM) и продлевается сам, без участия человека, пока не истечёт срок; статус и оставшийся срок видны в панели, с предупреждением, если доступ скоро истечёт; отключить аккаунт можно в один клик.
- **Безопасность API** — ограничитель частоты запросов на все внутренние API (отдельный, более строгий лимит на вход в панель — защита от подбора пароля); утилита проверки подлинности вебхуков готова заранее для будущей Instagram-интеграции.
- **Подключение Claude** — администратор вводит API-ключ в панели; ключ хранится зашифрованным и проверяется рабочим запросом к Anthropic API, статус проверки виден в панели.
- **Сбор данных Instagram** — фоновый опрос подключённых аккаунтов: медиа (посты/Reels/сторис), комментарии, метрики/инсайты (включая прирост подписчиков, тапы по ссылке и демографию аудитории при 100+ подписчиков) собираются автоматически с разной периодичностью, без дублей; статус последнего опроса виден в панели.
- **Дашборд аналитики** — раздел «Аналитика» с переключением между подключёнными аккаунтами: графики динамики по каждой собираемой метрике аккаунта, сортируемая таблица публикаций и демография аудитории (возраст/пол, топ стран); при малом объёме данных — понятные заглушки вместо пустых графиков.
- **Мониторинг доступности** — автоматическая проверка сайта каждые 10 минут, email-уведомление при простое.

Полная дорожная карта (аналитика Instagram, AI-агент для директа и комментариев, приём и учёт заявок, публикация контента, сайт турагентства) — в интерактивной дорожной карте: **[roadmap-platform.oresh.in](https://roadmap-platform.oresh.in)**.

## Стек

Next.js (App Router) + TypeScript + Prisma/PostgreSQL + NextAuth, задеплоено на DockHost. Подробности — в [ARCHITECTURE.md](ARCHITECTURE.md).

## Запуск локально

```sh
cd app
cp .env.example .env   # заполнить значения
docker compose up
```

Приложение поднимется на `http://localhost:3001` (Postgres — на `5433`).

## Безопасность

Секреты — только в `.env` (не в коде и не в коммитах). Токены доступа Instagram шифруются перед сохранением в базу. Отдельный API-ключ Claude под проект с лимитом расходов. Правила подробнее — в [CONTRIBUTING.md](CONTRIBUTING.md).

## Статус

Проект в активной разработке (этап MVP). Первый заказчик — турагентство; после того как вертикаль будет полностью построена, платформа станет применимой шире.
